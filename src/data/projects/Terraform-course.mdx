## **Overview**
A practical demonstration repository created for an introductory presentation on Terraform Infrastructure as Code (IaC). This project contains a series of incremental labs and complete sample configurations, guiding you from setting up your environment to provisioning your first infrastructure resources. It's designed to be a clear, executable blueprint for understanding Terraform's basic syntax and operational lifecycle.

## **Features**

✅ **lab01** - How to create a text file in a local machine.  
✅ **lab02** - How to create a text file in a local machine using variables.  
✅ **lab03** - How to use more than one provider.  
✅ **lab04** - How to use the Terraform console command fetching information from the variables.  
✅ **lab05** - How to create resources on AWS.  
✅ **lab06** - How to create resources in Azure.
